This is a Todo App to help organize your todos

How it works

  • User creates an account
  • User Logs in account
  • User creates Todo category eg. Study
  • User adds a Todo

Functionalities

Filters

Users can filter their todos according to add date and categories

Delete

Users can delete created todos

Update

Users can update todos to completed or not completed
